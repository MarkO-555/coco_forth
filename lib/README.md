## `trig.fs`

**Requires:** kernel primitives C@, *, +, -, /MOD, NEGATE, RSHIFT, CMOVE, sin-data, DUP, DROP, SWAP, OVER, @, !, <, >, 0=, IF, ELSE, THEN
